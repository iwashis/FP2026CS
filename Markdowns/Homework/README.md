# Homework Submission Guidelines

## Public Repository & Submission

All work must be maintained in a single **public GitHub repository** named **`Haskell-Course-2026`**.

The submission timestamp is determined by the **last commit on the `main` (or `master`) branch** before the deadline.

---

## Repository Structure

Your repository must follow the directory structure below. Failure to comply may result in a **-2 point penalty per exercise**.

```
/Haskell-Course-2026
│
├── /homeworks
│   ├── /hw1
│   │   └── Solution.hs
│   ├── /hw2
│   │   └── Solution.hs
│   ├── /hw3
│   │   └── Solution.hs
│   ├── /hw4
│   │   └── Solution.hs
│   └── /hw5
│       └── Solution.hs
│
└── /project
    ├── /src
    │   └── Main.hs (and other modules)
    ├── README.md   (Documentation and build instructions)
    ├── stack.yaml  (or package.yaml / project-name.cabal)
```

---

## Late Policy

### Homeworks
Late homework submissions are **not accepted**. The version of your code available in the repository on **Sunday at 23:59** of the due week will be graded.
